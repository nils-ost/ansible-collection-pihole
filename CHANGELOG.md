## v1\.0\.0

<a id="release-summary"></a>

### Release Summary

This is the first proper release of <code>nils_ost\.pihole</code> collection on 2026\-04\-01\.
It\'s mainly a carryover from other projects with some slight improvements\.

<a id="new-modules"></a>

### New Modules

- nils_ost\.pihole\.config \- read or write PiHole config key\.
- nils_ost\.pihole\.domain \- manage domain deny or allow\.
- nils_ost\.pihole\.list \- lists all elements\.

<a id="new-roles"></a>

### New Roles

- nils_ost\.pihole\.configure \- configures basic PiHole features\.
- nils_ost\.pihole\.install_with_docker \- installs PiHole within docker\.
